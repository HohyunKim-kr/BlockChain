메모리, CPU

메모리 <-- 데이터저장


Call Stack

```js
const a = 10;
function fn() {
  console.log(c);
  let c = 10;
}
let b = fn();
const c = 10;
console.log(a + b);

fn();
```

## 호이스팅

### TDZ (Temporal Dead Zone)

```js
function fn() {
  console.log(a);
  const a = 10;
}
fn();

// console.log(a)
// const a = 10
```

문제
`실행` 에 대한 문제입니다.

```js
function inner() {
  console.log(3, this);
  const outer = () => {
    console.log(2, this);
    const hello = () => {
      console.log(4, this);
    };
    hello();
  };
  outer();
  return () => {
    console.log(1, this);
  };
}
const fn = inner.call({ name: "ingoo" });
fn();
```
